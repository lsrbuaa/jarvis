# Hey! Jarvis

面向 OPC 办公人群的感知式 AI 效率工具原型。

Hey! Jarvis 用手机感知真实生活，用 AI 编排个人事务，让用户以更低成本管理工作、健康、生活和阶段性记忆。它不是停留在聊天框里的 AI 助手，而是把日程、手机传感器、生活日志和数字角色组织成一个可执行、可记录、可回看的个人效率入口。

![Jarvis product preview](jarvis_visual_final.png)

## 项目定位

在 AI 时代，每个人都期待拥有一个像 Jarvis 一样的个人智能管家：它不只回答问题，还能理解日程、感知状态、记录生活，并在合适时刻主动帮用户处理事务。

Hey! Jarvis 的第一版选择从 OPC 办公场景切入，使用用户手边已经拥有的手机，构建一个低输入成本的个人效率系统。它通过统一 Agent 入口、手机传感器和 AI 生成能力，帮助用户完成四类核心任务：

| 核心服务 | 解决的问题 | 产品能力 |
| --- | --- | --- |
| 日程编排 | 任务、会议和提醒分散，手动录入成本高 | 通过自然语言创建 todo、task、提醒和日程，并统一进入时间线 |
| 办公健康提醒 | 久坐、坐姿、光照和疲劳影响办公效率 | 使用摄像头、IMU、光线等手机能力进行轻量感知和低打扰提醒 |
| 生活日志 | 日记、饮食和日常片段难以持续记录 | 由手机感知、用户补充和 AI 整理共同生成可回看的生活素材 |
| 时光记忆 | 不同阶段的自己难以保存和回看 | 将照片生成带有记忆、人设和对话上下文的数字角色 |

## 核心创意

传统 AI Agent 最大的问题是只能等待用户输入，传统健康或效率硬件又需要额外设备。Hey! Jarvis 的核心创意是：

> 把手机变成个人 Agent 的感知器官，把日程和日志变成 Agent 的工作上下文，把照片生成的电子角色变成阶段性记忆容器。

它不是单纯创建一个电子宠物，也不是给日历套一层 AI 聊天。它的价值在于让用户不需要额外硬件，就能让 AI 开始理解自己的办公状态和生活节奏。

## 产品结构

当前原型是一个轻量 Android WebView 应用，主界面围绕 5 个标签页展开，全局 Agent 输入常驻底部，负责把一句话分发到正确的页面和动作。

| 标签页 | 回答的问题 | 放进去的功能 |
| --- | --- | --- |
| 今日 | 我现在该做什么？ | 下一步行动、今日工作负荷、健康节奏、专注/补水/站立/日报快捷动作 |
| 计划 | 我的任务和会议怎么排？ | To-do、Calendar、Focus block、健康提醒统一时间线 |
| 工作 | 我现在如何进入办公状态？ | 专注/会议/演示/文档模式、手势识别、办公指令队列 |
| 健康 | 身体和能量还能不能撑住工作？ | 饮水、站立、久坐、光照、饮食、压力和休息建议 |
| 记忆 | 今天沉淀了什么？ | 图文日记、情绪记录、数字角色状态、日报/周报、Agent 对话摘要 |

## 演示路径

1. 早上开工：对 Jarvis 说“今天上午帮我安排一下”，Jarvis 解析会议、任务和提醒，生成 todo 和 task，并展示今天的办公时间线。
2. 工作进行中：Jarvis 通过手机摄像头和 IMU 判断坐姿与静止时长，发现久坐或坐姿异常时给出低打扰提醒。
3. 午间生活记录：用户拍摄午餐，Jarvis 记录餐食、估算热量，并把生活片段归入当天日志。
4. 时光记忆封存：用户上传过去某个阶段的照片，Jarvis 生成数字角色，并为它创建独立 Agent workspace，用来保存人设、记忆和对话。

## 差异化优势

| 对比对象 | 传统方式 | Hey! Jarvis |
| --- | --- | --- |
| AI 助手 | 主要停留在对话框 | 连接日程、传感器、日志和数字角色 |
| 日程工具 | 手动创建事项，信息孤立 | Agent 一键创建，并统一进入日历视图 |
| 健康硬件 | 依赖手环、外设或主动打卡 | 使用手机传感器完成轻量监控 |
| 日记工具 | 依赖用户主动记录 | 传感器旁观记录 + 用户补充 + AI 整理 |
| 照片相册 | 只能回看静态照片 | 生成承载阶段记忆的数字角色 |

## 赛道匹配

Hey! Jarvis 面向 OPC 创意工具赛道，重点不只是“好玩”，而是用新的输入方式和记录习惯提升个人效率：

- 提高日程管理效率：统一 Agent 入口创建 todo、task 和日历事项。
- 提高办公健康维护效率：基于手机传感器监控久坐、坐姿和环境状态。
- 提高生活记录效率：旁观式记录生活片段，支持餐食上传和热量记录。
- 提高长期复盘效率：用数字角色封存不同时期的用户记忆和人设。
- 降低使用门槛：不依赖额外硬件，只使用手机已有能力。

## 愿景

Hey! Jarvis 不是把科幻里的智能管家一次性搬进现实，而是从 OPC 用户最真实的工作与生活管理问题开始，让手机先成为每个人的第一台 AI 管家原型。
