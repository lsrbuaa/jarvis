# Jarvis Android Prototype

Jarvis 是面向 OPC 办公人群的手机端办公助手原型，基于 Nori 的轻量 Android WebView 壳重构，并复用 Pet 项目的电子宠物、传感器和 Agent 产品思路。

## 交差版产品结构

一句话：Jarvis 是面向 OPC 办公人群的低摩擦办公闭环，把今日行动、工作模式、健康节奏和个人记忆收在一个手机端助手里。

底部导航只保留 5 个标签页，全局 Agent 输入常驻底部，不单独占一个标签页。

| 标签页 | 回答的问题 | 放进去的功能 |
| --- | --- | --- |
| 今日 | 我现在该做什么？ | 下一步行动、今日工作负荷、健康节奏、专注/补水/站立/日报快捷动作 |
| 计划 | 我的任务和会议怎么排？ | To-do、Calendar、Focus block、健康提醒统一时间线 |
| 工作 | 我现在如何进入办公状态？ | 专注/会议/演示/文档模式、手势识别、办公指令队列 |
| 健康 | 身体和能量还能不能撑住工作？ | 饮水、站立、久坐、光照、饮食、压力和休息建议 |
| 记忆 | 今天沉淀了什么？ | 图文日记、情绪记录、HXD 状态、日报/周报、Agent 对话摘要 |

### 收起或暂不做

- 不做完整项目管理系统，只做个人日程和行动项。
- 不做独立桌宠养成主线，HXD 只作为状态反馈和记忆拟人化表达。
- 不把饮食、日记、手势、传感器分别做成主标签，它们归入健康、记忆和工作。
- 不做独立 AI 标签页，Agent 是底部输入和分发器，负责把一句话落到正确页面。
- 复杂模型配置和调试入口放进侧边菜单或设置弹窗，不进入主导航。

### 演示路径

1. 进入「今日」看下一步建议，一键开始专注。
2. 进入「计划」查看会议、任务和专注块的统一时间线。
3. 进入「工作」切换会议/演示模式，展示手势到办公指令。
4. 进入「健康」记录补水、站立或饮食，体现办公健康节奏。
5. 进入「记忆」生成日报，展示 Jarvis 如何把一天沉淀为长期记忆。
